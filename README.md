Коллекция проектов, демонстрирующая навыки промышленного программирования на Python. Здесь собраны работы от алгоритмов до архитектурных паттернов.

## 🔥 Ключевые компетенции в репозитории:

*   **Архитектура и ООП**: Использование `dataclasses` (настройки, To-Do list), наследование от встроенных типов `str` и `list` (`Smart_string`, `Limited_list`).
*   **Функциональное программирование**: Продвинутые декораторы, цепочки декораторов, работа с `*args` и `**kwargs`.
*   **Оптимизация**: Применение мемоизации (`lru_cache`) для ускорения рекурсивных алгоритмов.
*   **Надежность кода**: Проектирование собственных систем исключений (`Custom Exceptions`) и глубокая валидация входных данных.
*   **Математические алгоритмы**: Геометрические вычисления, работа с последовательностями и рекурсией.

## 🛠 Стек технологий:
*   **Language**: Python 3.10+
*   **Libraries/Tools**: `functools`, `dataclasses`, `datetime`, `typing`.
